# Overview

This repository provides open, practical assets for the AI Inference Reference Architecture. Assets are grouped by composition size (OPG/XOC) and profile (minimal/distributed). Each variant includes:

- connectivity-map.csv — cabling and port mapping
- bom.yaml — bill of materials
- wiring.yaml — Hedgehog Wiring CRDs
- vpc.yaml — Hedgehog VPC CRDs
- diagrams/ — matching connectivity visuals
