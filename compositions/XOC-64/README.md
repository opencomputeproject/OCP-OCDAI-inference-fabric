# XOC‑64 — Variants Overview (Inference)

XOC‑64 wraps a single OPG‑64 in the XOC composition layout, adding the spine/aggregation layer, distributed gateway infrastructure, and external connectivity that the OPG building block does not supply on its own.

The compositions in this repository are designed to work with air or liquid cooling at any density. Cooling medium and rack density are physical deployment choices that do not affect the network topology.
